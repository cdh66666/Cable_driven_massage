# 轨迹图片生成器

本工具是本地 Windows 图形界面，用来把图片快速处理成硬件可绘制的多段轨迹。

常用位图格式优先支持：PNG、JPG/JPEG、BMP、TIFF、WebP、GIF 首帧。SVG/PDF 这类矢量或文档格式请先另存为 PNG。

## 启动

双击仓库根目录的 `run_trajectory_studio.bat`，或在 PowerShell 里运行：

```powershell
python tools\trajectory_studio.py
```

## 输出

- `轨迹 TXT`：和网页 `/path` 接口一致，一行一段轨迹，点格式为 `x,y;x,y;...`，单位 mm，范围为 `0..150`。导出时会自动压到板子安全点数，适合直接导入网页端。
- `JSON`：多段轨迹数组，方便二次处理。
- `预览 PNG`：白底黑线预览图，带 150x150 mm 边框。

## 和板子网页端连通

1. 在本工具打开图片，等待自动转换完成。
2. 方式 A：USB 线连接板子，在 `USB端口` 填 `COM52`，点 `USB上传轨迹`。上传成功后，网页端点 `开始`，或者串口发 `webstart`。
3. 方式 B：点 `导出轨迹 TXT`，手机或电脑连接 `MassageDraw` 热点后打开网页，点 `导入轨迹` 选择这个 TXT，再点 `开始`。
4. 方式 C：电脑直接连接 `MassageDraw` 热点，点本工具里的 `WiFi上传到板子`，上传成功后在网页端点 `开始`。

## 处理建议

- 照片/壁纸：优先用 `整图素描`，会保留整张图的边缘和灰度水平排线，更接近铅笔素描照片。
- 只想画人物主体：用 `照片主体线稿`，会尽量过滤背景。
- 细线稿、手绘图：优先用 `深色线稿` 或 `自动`。
- 粗黑线、字、图标：优先用 `黑色区域轮廓`，会更像原图轮廓。
- 照片里的物体轮廓：优先用 `物体外轮廓`。
- 细节太乱：增大 `去噪面积`、`最短轨迹(px)`、`平滑容差(mm)`。
- 画出来太粗糙：减小 `点间距(mm)`、`平滑容差(mm)`，或增大 `处理分辨率`。
- 硬件运动太慢：增大 `点间距(mm)` 或降低 `最多点数`。
- `中心线(慢)` 会做纯 Python 细化，只有在非常干净的线稿上才建议用。

轨迹会自动缩放居中到硬件 150x150 mm 区域内，并留出可调边距。
